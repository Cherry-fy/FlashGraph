See the License for the specific language governing permissions and limitations under the License. This class defines an interface of a I/O scheduler for requests generated by user tasks. It is only used in the page cache. Right now, an I/O scheduler is associated with an I/O instance and can only schedule I/O requests issued by the I/O instance.
