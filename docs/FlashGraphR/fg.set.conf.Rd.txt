Reconfigure FlashGraphR

Description:

     This reconfigures FlashGraphR with the settings in the
     configuration file. The configuration file contains a list of
     key-value pairs. Each line in the file is a key-value pair in the
     form of "key_name=value".

Usage:

     fg.set.conf(conf.file)
     
Arguments:

conf.file: The configuration file.
